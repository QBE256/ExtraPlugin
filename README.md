# ExtraPlugin
5chでだけ晒していたやつとか、ニッチそうなのをいろいろまとめる

カスタムパラメータは一つのJSONの中にパラメータをまとめる必要があります。

よって、下記のように記載する必要があります。 

※ 外枠の`{}`については各種スクリプトファイルの例からは省略していますので注意してください
```
{
  <スクリプトAのカスタムパラメータ>,
  <スクリプトBのカスタムパラメータ>,
  <スクリプトCのカスタムパラメータ>,
  ...
  <スクリプトZのカスタムパラメータ>
}
```
